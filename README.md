A terminal-based Python BlackJack game that can be played directly in the command line.

This module provides functionality to initialize and run an interactive game experience
within a terminal environment. Players can engage with the game through command-line inputs
and receive text-based outputs.

To start the game:

1. Ensure Python is installed on your system
2. Navigate to the directory containing this file
3. Run the command: `python main.py`
4. Follow the on-screen prompts to play

Requirements:

- Python 3.6 or higher
- Standard library modules (no external dependencies required)

Features:

- Interactive terminal-based gameplay
- Text-based user interface
- Command-line input handling
